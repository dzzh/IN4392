Dependency to be installed : http://code.google.com/p/google-visualization-python/


Monitoring tool usage:

To start the tool, run server.py script.

go to http://0.0.0.0:8080/ and you will see a form.  Enter an environment id that is present in aws.config file.  Press Submit.

Next, you will see a list of links that represent the running instances.  You can click on any one of them.

You should see next a graph showing CPUUtilization for the selected instance, from launch time to current moment.  
If it not appears immediately, wait 1-2 seconds.
